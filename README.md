# Rander

Just want to generate a string from mostly of charset which will get a shorter string. 

Now, just use Time.now + rand(32) to be a code

## Installation

Add this line to your application's Gemfile:

    gem 'rander'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rander
